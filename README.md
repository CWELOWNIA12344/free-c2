# Moja Statyczna Strona

Prosta statyczna strona HTML z plikiem wideo, gotowa do szybkiego deployu na [Vercel]([https://pexen-c2.vercel.app/]).  
Strona działa zarówno lokalnie, jak i po wrzuceniu na serwer Vercel.

---

## Struktura katalogów

- `index.html` – strona główna (Twoja obecna `C2.html`, zmień nazwę)  
- `public/Baciata.mp4` – pliki statyczne (np. wideo)  
- `vercel.json` – konfiguracja deployu  
- `README.md` – ten plik

---

## Uruchamianie lokalnie

1. Otwórz `index.html` w przeglądarce.  
2. Lub, jeśli masz zainstalowany [Vercel CLI](https://vercel.com/docs/cli):

```bash
vercel dev
